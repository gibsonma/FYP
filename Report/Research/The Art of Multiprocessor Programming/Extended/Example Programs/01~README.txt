These files contain complete, working versions of many of the data structures and algrorithms used in "The Art of Multiprocessor Programming".

The Java programs are organized as NetBeans(tm) projects. If you don't use NetBeans, then you can find the source programs in the src folder, and test drivers in the test folder.

In the Appendix1 folder, the C# examples are organized as a Visual Studio(tm) project.
